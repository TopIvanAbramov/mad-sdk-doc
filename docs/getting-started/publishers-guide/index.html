# Базовая инструкция для паблишеров

Инструкция по интеграции рекламы через платформу MADS.

# 1. Базовые понятия

Термин Описание
Рекламный плейсмент (пад) Конкретное место для показа рекламы на вашей площадке. Идентифицируется по pad_id
Рекламный формат Способ отображения рекламы (баннер, видео, карусель)
Рекламный креатив Конкретный рекламный материал для показа
Рекламная интеграция Способ подключения и отображения рекламы (SDK или API)

# 2. Регистрация в системе

  1. Зарегистрируйтесь через B2B-портал по номеру телефона
  2. Заполните регистрационную информацию
  3. Сообщите менеджеру о регистрации для активации доступа

# 3. Настройка плейсментов

  1. Войдите в кабинет паблишера
  2. Перейдите в раздел управления плейсментами
  3. Создайте новый плейсмент:
    • Укажите понятное название
    • Выберите платформу (Android, iOS, Web, Other)
    • Настройте поддерживаемые форматы

# 4. Технические интеграции

Платформа Действия
Мобильные приложения Установите MADS SDK, добавьте коды вызова рекламы с указанием pad_id, настройте события показа
Веб-сайты Подключите SDK в код сайта, внедрите коды вызова рекламы на нужные страницы, укажите pad_id для каждого места показа
Видео-плееры Настройте VAST/VPAID интеграцию, добавьте URL вызова рекламы в конфигурацию плеера

Подробнее: Способы интеграции

# 5. Работа с форматами

  • Выберите поддерживаемые форматы при создании плейсмента
  • Для нестандартных решений обратитесь к менеджеру
  • Используйте тестовые плейсменты для проверки отображения

# 6. Запуск рекламы

  1. Завершите техническую настройку
  2. Сообщите менеджеру о готовности
  3. Дождитесь добавления падов в рекламные пакеты

# 7. Мониторинг

  • Отслеживайте статистику по показам и кликам
  • Анализируйте данные с разбивкой по плейсментам
  • При необходимости корректируйте настройки

# Важные рекомендации

  • Правильно разделяйте плейсменты для детальной аналитики
  • Тестируйте каждый формат перед запуском реальной рекламы
  • Поддерживайте актуальную версию SDK
  • Своевременно сообщайте о технических проблемах менеджеру
