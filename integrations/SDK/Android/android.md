# Android

MadSDK — омниканальная платформа персонализации. Инструменты Mad SDK на основе больших данных адаптируют сайт, приложение под потребности каждого пользователя с целью оптимизации конверсии, роста выручки и других продуктовых метрик.

Mad SDK включает в себя:

Сегментацию и таргетинг
Товарные рекомендации
Персонализацию баннеров, контента, навигации и любых других визуальных элементов
А/Б тестирование
Аналитику

#Варианты интеграции
В Mad SDK  существует три варианта интеграции:

Web — Установка JS скрипта на сайт.
SDK — интеграция в мобильное приложение
