# Быстрый старт

Справочник по API рекламной ручки MADS.

## Авторизация

API не требует авторизации. Для запроса рекламы используется `pad_id` плейсмента.

## Базовый URL

```
https://mads-serve.magnit.ru/
```

## Идентификация пользователей

В query-параметрах запроса необходимо передавать `user_id` — идентификатор пользователя на вашей стороне. Это позволяет:

- персонализировать рекламу под пользователя
- учитывать статистику по пользователям
- связывать события показа и клика с конкретным пользователем

## Обработка ошибок

1. Установите таймаут для ожидания ответа от сервера.
2. Если полученный ответ имеет статус, отличный от 200, выполните одно из следующих действий:
   - Не отображайте ничего.
   - Отобразите заглушку, например, стандартный баннер.
