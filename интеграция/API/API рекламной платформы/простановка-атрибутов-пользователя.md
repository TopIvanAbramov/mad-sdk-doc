# Простановка атрибутов пользователя

Передача атрибутов пользователя для таргетирования и персонализации рекламы.

## Назначение

Атрибуты позволяют рекламной системе подбирать более релевантные объявления на основе:
- пола, возраста пользователя
- интересов и поведения
- контекста (экран, категория контента)

## Параметры таргетирования

При запросе рекламы можно передать словарь таргетингов:

| Ключ | Пример | Описание |
|------|--------|----------|
| `gender` | `female` | Пол пользователя |
| `age_group` | `25-34` | Возрастная группа |
| `screen` | `main` | Текущий экран |
| `category` | `food` | Категория контента |

## Передача в запросе

Атрибуты передаются в теле или query-параметрах запроса получения рекламы. Точный формат зависит от реализации API.

Подробнее о таргетировании: [Таргетирование](../../../concepts/targeting.md)
