# Отправка взаимодействия с рекламой

Отправка событий взаимодействия пользователя с рекламой: показ, клик, конверсия.

## Назначение

События необходимы для:
- учёта статистики показов и кликов
- расчёта эффективности рекламных кампаний
- оптимизации таргетирования

## Типы событий

| Событие | Описание |
|--------|----------|
| `impression` | Показ рекламы |
| `click` | Клик по рекламе |

## Пиксели (Statistic)

Список URL для отправки событий приходит в ответе на запрос рекламы в массиве `statistics`. Каждый элемент содержит:
- `url` — URL для вызова
- `type` — тип события (impression, click и др.)
- `value`, `pvalue`, `viewablePercent` — опциональные параметры

Подробнее: [Модель Statistic](../models/pereispolzuemye-modeli.md#statistic)

## Пример

После показа рекламы вызовите URL из `statistics` с типом `impression`. После клика — URL с типом `click`.
