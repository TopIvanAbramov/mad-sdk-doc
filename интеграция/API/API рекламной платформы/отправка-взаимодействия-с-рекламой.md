# Отправка взаимодействия с рекламой

Отправка событий взаимодействия пользователя с рекламой: показ, клик, конверсия.

## Назначение

События необходимы для:
- учёта статистики показов и кликов
- расчёта эффективности рекламных кампаний
- оптимизации таргетирования

## Типы событий

| Событие | Описание | Когда отправлять |
|--------|----------|-----------------|
| `impression` | Показ рекламы | При достижении порога видимости элемента. По умолчанию — **50%** площади. Если в `Statistic` задан `viewablePercent` — использовать его значение. |
| `click` | Клик по рекламе | При нажатии на кнопку с диплинком (`button.type == "cta"`) либо на интерактивный элемент (обложку сторис). |
| `promocodeClick` | Клик по промокоду | При нажатии на кнопку с промокодом (`button.type == "customAction"`, `button.subType == "promocodeCopy"`). |

## Как отправлять события?

Для отправки событий необходимо выполнить GET запрос на ссылку пикселя. Пиксель — это URL, описывающий событие с защищёнными параметрами. Каждый пиксель необходимо прогружать **один раз** в рамках одного ответа.

Список URL для отправки событий приходит в ответе на запрос рекламы в массиве `statistics`. Каждый элемент содержит:
- `url` - URL для вызова
- `type` - тип события (`impression`, `click`, `promocodeClick` и др.)
- `value`, `pvalue`, `viewablePercent` - опциональные параметры

Подробнее: [Модель Statistic](../models/pereispolzuemye-modeli.md#statistic)

## Где находятся пиксели

Пиксели присутствуют на разных уровнях модели ответа - в зависимости от рекламного формата.

### Stories

| Уровень | Тип пикселя | Триггер |
|---------|-------------|---------|
| `AdResponse.statistics` | `impression` | Блок сторис достиг порога видимости |
| `Creative.statistics` | `impression` | Обложка сторис достигла порога видимости |
| `Creative.statistics` | `click` | Нажатие на обложку сторис |
| `StorySlide.statistics` | `impression` | Слайд сторис достиг порога видимости |
| `StorySlide.statistics` | `click` | Нажатие на кнопку с диплинком (type `cta`) на слайде |
| `StorySlide.statistics` | `promocodeClick` | Нажатие на кнопку с промокодом на слайде |

### Multiformat

| Уровень | Тип пикселя | Триггер |
|---------|-------------|---------|
| `AdResponse.statistics` | `impression` | Блок баннера достиг порога видимости |
| `Creative.statistics` | `impression` | Слайд баннера достиг порога видимости |
| `Creative.statistics` | `click` | Нажатие на кнопку с диплинком (type `cta`) на слайде баннера |

### ModalWindow

| Уровень | Тип пикселя | Триггер |
|---------|-------------|---------|
| `Creative.statistics` | `impression` | Модальное окно показано пользователю |
| `Creative.statistics` | `click` | Нажатие на кнопку с диплинком (type `cta`) в модальном окне |
| `Creative.statistics` | `promocodeClick` | Нажатие на кнопку с промокодом |
