# API интеграция

Интеграция рекламы через REST API без использования SDK. Подходит для кастомных решений, видео-плееров (VAST/VPAID) и серверных приложений.

## Особенности

- **Без авторизации** — запросы выполняются по `padId` плейсмента
- **padId** — обязательный параметр, идентификатор рекламного плейсмента из кабинета паблишера

## Быстрый старт

1. Создайте плейсмент в кабинете паблишера и получите `padId`
2. Выполните GET-запрос к API с параметром `padId`
3. Обработайте ответ согласно [моделям данных](models/index.md)
