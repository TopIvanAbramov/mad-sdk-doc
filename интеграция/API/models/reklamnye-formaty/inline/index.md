InLine реклама — это встраиваемые рекламные блоки внутри контента приложения. Поддерживаемые типы: **Stories**, **Banner**, **Clickout**.

Возвращаемый тип рекламного блока регулируется на стороне админки.

Подробнее о форматах — в разделе [Рекламные форматы](../../../../../concepts/ad-formats.md).


## Форматы

- [Multiformat](multiformat.md) — баннеры
- [Stories](stories.md) — вертикальные карточки со слайдами
