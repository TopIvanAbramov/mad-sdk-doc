# Интеграция

Способы подключения рекламы платформы MADS к вашей площадке.

## Варианты интеграции

| Способ | Описание | Документация |
|--------|----------|--------------|
| **Mobile SDK** | Android и iOS приложения | [Android SDK](Mobile SDK/Android SDK/index.md) |[iOS SDK](Mobile SDK/iOS SDK/index.md) |
| **Web SDK** | Веб-сайты | [Web](Web/index.md) |
| **API** | Прямые запросы по pad_id | [API интеграция](API/index.md) |

## Mobile SDK

Готовые SDK для мобильных платформ с поддержкой In App Message и InLine форматов.

- [Android SDK](Mobile SDK/Android SDK/index.md) — быстрый старт, In App, InLine, отладка
- [iOS SDK](Mobile SDK/iOS SDK/index.md) — быстрый старт, In App, InLine, отладка

## Web SDK

Интеграция через JavaScript для веб-сайтов. [Подробнее](Web/index.md)

## API

Интеграция без SDK: запросы к API с указанием `pad_id` плейсмента. Подходит для кастомных решений и видео-плееров.

- [Начало работы с API](API/index.md)
- [Модели данных](API/models/index.md)
- [API Reference](API/api-reference.md) -->
