# :icon-device-mobile: Android SDK

Руководства по интеграции MADS SDK в Android-приложение.

## Требования

| Параметр | Значение |
|----------|----------|
| minSdk | 26 |
| targetSdk | 35 |
| compileSdk | 35 |
| Kotlin | 2.1.21 |
| Compose (BOM) | 2025.09.01 |

## Содержание

- [Быстрый старт](quickstart.md) — Установка, инициализация, идентификация пользователя
- [Рекламные форматы](<Рекламные форматы/index.md>) — In App Message (ModalWindow), InLine (Multiformat)
- [Отладка](debugging.md) — Тестирование и отладка интеграции

Пример интеграции SDK можно посмотреть в [репозитории на GitHub](https://github.com/magnit-tech/mads-android-sdk).
