# Changelog

## v0.0.1
- Первый релиз MADS SDK
- Поддержан рекламный формат ModalWindow


## v0.0.2 (comming soon)

- UI конфиг полностью вынесен на бекенд. Кастомизация будет доступна в админ панели

- Отмена загрузки рекламы
