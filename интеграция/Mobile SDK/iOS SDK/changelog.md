# Changelog

## v0.0.1
- Первый релиз MADS SDK
- Поддержан рекламный формат ModalWindow


## v0.0.4

- UI конфиг полностью вынесен на бекенд. Кастомизация будет доступна в админ панели
- Event разделены на 2 части: Actions - действия пользователя, которые необходимо обработать, Event - рекламные события, не требующие обработки
- Отмена загрузки рекламы
- Поддержаны загрузка, подписка на ивенты через async / await
