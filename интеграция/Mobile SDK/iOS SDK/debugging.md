# Отладка интеграции — iOS

## Тестовый рекламный креатив

Для отладки интеграции можно использовать отладочный рекламный креатив:

```swift
loader.load(
    padId: 67,                    // отладочный padId 
    isDebugCreativeEnabled: true // включаем загрузку тестового креатива
)
```

## Дебаг-логи

Для получения подробных логов включите режим отладки:

```swift
MadsSDK.isDebugLogsEnabled = true
```

> [!TIP]
> Тестовый креатив позволяет проверить корректность интеграции без реальных рекламных кампаний.
