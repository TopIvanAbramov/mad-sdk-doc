# Stories


![|300x200](./stories.png){height="100"}

- Обложка stories отображается с акцентной обводкой, если она не просмотрена.

- Слайды передают статус просмотра в обводку обложки
После просмотра всех слайдов сторис, обводка исчезает с обложки, сторис считается просмотренной 

- Просмотренные stories перемещаются в конец списка, сохраняя изначальный порядок внутри своей группы.

- Непросмотренные stories отображаются первыми, затем — просмотренные.

- Очередь обновляется после каждого полного или частичного просмотра.

- Нажатие на обложку открывает stories с первого слайда.

- Если просмотрен хотя бы один слайд, то открывается, начиная с непросмотренного слайда
