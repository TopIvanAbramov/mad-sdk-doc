# :icon-graph: Сбор статистики

MADS SDK автоматически собирает статистику показов, кликов и конверсий без дополнительных действий со стороны интегратора. Все ключевые метрики передаются на рекламную платформу.

Подробную статистику по показам и кликам можно посмотреть в админ панели.

Для верификации корректности сбора данных можно подключить пиксели статистики от поддерживаемых верификаторов в админ‑панели MADS.
