# Способы интеграции

MADS предлагает три способа интеграции рекламы на вашу площадку.

## Mobile SDK

Интеграция через нативные SDK для мобильных приложений.

| Платформа | Описание | Статус |
|-----------|----------|--------|
| [Android SDK](../../интеграция/mobile-sdk/android-sdk/) | Интеграция в Android-приложение | ✅ Доступно |
| [iOS SDK](../../интеграция/mobile-sdk/ios-sdk/) | Интеграция в iOS-приложение | ✅ Доступно |

Подходит для: приложений на Android и iOS. SDK обрабатывает загрузку рекламы, отображение и сбор статистики.

## Web SDK

Интеграция через JavaScript для веб-сайтов.

| Платформа | Описание | Статус |
|-----------|----------|--------|
| [Web SDK](../../интеграция/Web/) | Установка JS-скрипта на сайт | 🔜 В разработке |

Подходит для: веб-сайтов и SPA.

## API

Прямая интеграция через REST API без использования SDK.

| Раздел | Описание |Статус |
|--------|----------|----------|
| [API интеграция](../../интеграция/api/) | Запросы по `pad_id`, структура ответа, модели данных |🔜 В разработке |

Подходит для: кастомных интеграций, видео-плееров (VAST/VPAID), серверных приложений.

**Особенности:** API не требует авторизации. Для запроса рекламы используется `pad_id` плейсмента.
