# Рекламные форматы

MADS SDK поддерживает два основных способа показа рекламы: **In App Message** и **InLine**. Каждый из них включает несколько форматов.

## In App Message

In App Message — это всплывающие окна внутри приложения, где можно разместить изображение, кнопку с диплинком или копирование промокода. Такой формат помогает привлекать внимание пользователей и повышать конверсии прямо в интерфейсе приложения поверх любого экрана.

### Modal Window

Modal Window — это формат рекламы, который представляет собой диалоговое окно, которое появляется поверх текущего экрана пользователя.

Возможности кастомизации:
- Цвета фона, кнопок и контента
- Скругления элементов
- Кастомные шрифты (в будущем)

Подробнее о кастомизации — в платформенных гайдах: [Android](../интеграция/Mobile SDK/Android SDK/in-app.md) | [iOS](../интеграция/Mobile SDK/iOS SDK/in-app.md)

## InLine

> [!NOTE]
> InLine-форматы находятся в процессе разработки.

InLine реклама — это встраиваемые рекламные блоки внутри контента приложения. Поддерживаемые типы: **Stories**, **Banner**, **Clickout**.

Возвращаемый тип рекламного блока регулируется на стороне админки. Для мобильного приложения встраивание идентично для разных форматов.

### Stories

Stories — это формат рекламы, который представляет собой вертикальные полноэкранные карточки с контентом (текст, изображения, видео), последовательно показываемые поверх текущего интерфейса приложения и переключающиеся свайпами или тапами пользователя. Аналог сторис в Telegram.

### Banner

Баннерная реклама — встраиваемый блок с изображением и ссылкой, размещаемый внутри списков и контента приложения.

### Clickout

Clickout — формат рекламы с переходом по клику на внешний ресурс или диплинк.
