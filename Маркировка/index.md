# :icon-tag: Маркировка рекламы

![|300x200](./marking-info.png){height="100"}

В соответствии с требованиями законодательства, каждый рекламный материал, показываемый через MADS SDK, помечается специальным лейблом «Реклама». Лейбл отображается поверх креатива и служит для информирования пользователя о том, что данный контент является рекламным.
По клику на лейбл открывается карточка рекламодателя с обязательной информацией:
- Название рекламодателя
- ОГРН
- ERID (идентификатор рекламного объявления)

При создании нового рекламного объявления в админ‑панели MADS необходимо заполнить следующие поля:
- Название рекламодателя
- ERID - уникальный идентификатор рекламного объявления, присваиваемый по правилам «Единого реестра интернет‑рекламы».
- ОГРН - основной государственный регистрационный номер юридического лица.

После сохранения данных система автоматически подставляет эту информацию в карточку рекламодателя, доступную пользователю по клику на лейбл «Реклама».
Дальнейших действий со стороны интегратора для отображения лейбла и карточки не требуется - все поведение реализовано на SDK‑уровне.
