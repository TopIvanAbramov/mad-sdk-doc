# :icon-tag: Маркировка рекламы

![|300x200](./marking-info.png){height="100"}

В соответствии с требованиями законодательства, каждый рекламный материал, показываемый через MADS SDK, помечается специальным лейблом «Реклама». Лейбл отображается поверх креатива и служит для информирования пользователя о том, что данный контент является рекламным.
По клику на лейбл открывается карточка рекламодателя с обязательной информацией:
- Название рекламодателя
- ОГРН
- ERID (идентификатор рекламного объявления)

Дополнительный действий со стороны интегратора для отображения лейбла и карточки не требуется - все поведение реализовано на SDK‑уровне.
