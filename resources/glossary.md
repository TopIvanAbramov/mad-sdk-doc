# Глоссарий

| Термин | Определение |
|--------|-------------|
| **MADS** | Омниканальная рекламная платформа для ритейлеров |
| **MADS SDK** | Библиотека для интеграции ресурсов ритейлера с платформой MADS |
| **padId** | Идентификатор места размещения рекламы (Placement Ad ID) |
| **In App Message** | Всплывающее рекламное окно поверх экрана приложения |
| **InLine** | Встраиваемый рекламный блок внутри контента приложения |
| **Modal Window** | Формат In App Message — диалоговое окно поверх текущего экрана |
| **Stories** | Формат InLine-рекламы — вертикальные полноэкранные карточки |
| **Banner** | Формат InLine-рекламы — встраиваемый блок с изображением и ссылкой |
| **Clickout** | Формат InLine-рекламы — переход по клику на внешний ресурс или диплинк |
| **Таргетинг** | Параметры «ключ–значение» для персонализации подбора рекламы |
| **Креатив** | Рекламное объявление (изображение, текст, кнопки) |
| **Дебаг-креатив** | Тестовое рекламное объявление для отладки интеграции |
| **Диплинк** | Ссылка для перехода в определённый раздел приложения |
| **ERID** | Электронный идентификатор рекламного объявления (для маркировки) |
| **ОГРН** | Основной государственный регистрационный номер рекламодателя |
